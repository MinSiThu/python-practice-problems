# Operations on Grade List

Question:

You have a list called grades containing the scores of five students: [85, 92, 78, 90, 88]. Perform the following operations:

1. Print the second student's score.
2. Change the third student's score to 80.
3. Add a new student with a score of 95.
4. Remove the score of the fourth student.
5. Calculate and print the average score of the students.

Give it a try and see if you can perform these operations on the list!
